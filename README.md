# anime-recommender
🌸 Ultimate Anime Recommender

A smart and interactive anime recommendation web app built using Streamlit that helps users discover anime based on search, filters, and viewing patterns.

🔗 Live App:https://ultimateanimerecommender.streamlit.app/


🚀 Features

1)✨ Smart Search with Fuzzy Matching
Handles typos using RapidFuzz
Example: “deat note” → Death Note

2)🎯 Personalized Recommendations
Suggests anime based on your search and also from your search history(which is stored)
Improves discovery experience over time

3)📊 Filter System
Filter anime by genre
Clean and interactive sidebar UI

4)🌟 Most Popular Anime Section
Based on user engagement (ScoredBy)

5)In 4 and 5 there's a "Details" button which displays details below

6)📖 Detailed Anime Info
Genre
Rating
Episodes
Studio
Airing details


🛠️ Tech Stack
Python
Streamlit
Pandas
NumPy
RapidFuzz


🧠 How It Works
Uses fuzzy string matching to find closest anime names
Filters dataset using Pandas
Tracks user searches to generate history-based recommendations
Displays results dynamically using Streamlit UI components
